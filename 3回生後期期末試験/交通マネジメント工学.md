供給側の施策
交通施設の能力を増強し増える需要に対応するアプローチ。ハード施策が中心
・道路や鉄道などを新しく作る、広げる
・車両などの技術開発を促す

需要側の施策
人々の移動行動そのものを調整・管理して、既存施設の効率的利用や環境負荷低減を図るアプローチ。ソフト施策が中心
・TDM（交通需要マネジメント）
・MM（モビリティ・マネジメント）

ハード施策
道路、鉄道などの物理的インフラを新設・改良・変更することで問題を解決する。即効性や強制力があるが、コストが大きく一度作ると変更が難しい側面がある

ソフト施策
既存の施設を使いつつ、制度の運用変更、情報提供、意識啓発などを通じて人々の態度・行動を変えることで問題を解決する。比較的小さなコストで柔軟な変更が可能。効果の確実性や持続性には工夫が必要。

パッケージ施策
ハードとソフト融合させたもの

社会的ジレンマ
長期的には公共的な利益を低下させるものの、
短期的な私的利益の増進に寄与する行為（非協力行動）
か
短期的な私的利益は低下してしまうものの、
長期的には公共的な利益の増進に寄与する行為（協力行動）
のいずれかを選択しなければならない社会状況

その対処法
・構造的方略：社会の制度に働きかける（態度追従型）
・心理的方略：個人の態度と行動に働きかける（態度変容型）

構造的方略の例
・物理的・ハード面での整備（Pull施策中心）
	・バスや鉄道（LRT等）の新規導入、複線化などの性能向上
	・歩道の拡幅、自動車道の整備
・制度・運用の変更（Push施策中心）
	・金銭的負担の変更：ロードプライシング、駐車料金の値上げ、公共交通料金の値下げ
	・規制による禁止・制限：特定ゾーンへの自動車流入規制
	・物理的な抑制：車線の削除・撤去、ハンプ設置による速度抑制

心理的方略の例
・事実情報提供法
	・公共交通マップの配布
	・クルマ利用によるカロリー消費やコストの比較情報の提供
・経験誘発法
	・ノーマイカーウィーク
	・バスや鉄道の体験乗車チケット配布
・コミュニケーション法
	・依頼法
	・アドバイス法
	・行動プラン法

構造的方略は即効性があるがそればかり用いると損得勘定のみで動くようになり、長期的な公衆性や倫理性を損なう副作用がある
→心理的方略を用いて「内発的動機づけ」に働きかけること、あるいは両者を適切に組み合わせることが重要視されている。

心理的方略の重要性
・公衆性（モラル）の破壊を防ぐ：報酬を与えて協力行動を促すと、報酬が無くなった途端に協力行動が以前より低下してしまうアンダーマイニング効果のリスクがある
・行動の持続性を確保する

心理的方略の難しさ
・即効性と確実性の低さ：呼びかけや働きかけを受け入れるかどうかは個人の自由意思に委ねられるため必ず行動が現れる保証がない、効果が現れるまで時間がかかる
・個人の利益（ジレンマ）との戦い：いくら意識に働きかけても短期的・私的な利益の誘惑は強力であるため、呼びかけだけでは実際の行動変容まで結びつけるんが難しい場合がある
・コミュニケーションコストの高さ：対話相手の一人ひとりに対する手間や人的コストがかかる

構造的方略は即効性があるが副作用（モラルの低下）があり、心理的方略は持続性があるが確実性に欠ける

・フレックスタイム、時差出勤・通学

・戸別訪問：大規模かつ個別的なコミュニケーションとして、各家庭を訪問して情報提供を行った。
・ノーマイカーウィーク：一斉にマイカー利用を控えるよう呼びかけるキャンペーン（経験誘発法）
・教育・啓発プログラム：バス停マップなどのグッズを用いて個々人に具体的な行動計画を立ててもらう取り組みや、バスや地下鉄の利用メリットを視覚的に伝えるマップの配布（事実情報提供法）など

・渋滞や混雑の解消
・CO2排出量の削減
・地方格差の是正

21世紀の課題：「質的」な交通政策

人々の価値観や文化と交通政策の関係

価値観の変化に交通政策も追従している。価値観は、20世紀では移動の速さ、効率性、物質的な充足が重要だったが、21世紀では「楽しさ」「心地よさ」「安心感」といった主観的幸福度が重要になっている

物理的な道路整備だけでなく、その空間がもつ意味や文化を作り替えることが政策の重要な要素になっている
単に「車を通す場所」から「人が集い、にぎわう場所」への転換
交通政策の成果指標は交通量だけでなく「街の賑わい」「人々の気分」という文化的・心理的側面におかれている

12/15

モビリティ・マネジメント（MM）とは、広義には、地域や都市の交通を人・組織・社会の活力を通して改善する取り組み。交通渋滞、公共交通の衰退、都市の郊外化といった課題に対し、人々の過度な自転車依存から公共交通や自転車の適切な利用への転換（モーダルシフト）を促すことを目的とする。

MMの要領は、交通に関わる人々の気持ちを変え、自発的な行動変容を引き出すことにある。具体的には、「公衆とのコミュニケーション」と「関係者との調整」が重要となる。公衆コミュニケーションには、チラシ配布、アンケート、ラジオ、学校教育などが用いられ、健康や環境への配慮といった情報を伝えることで意識変容を促す。一方、関係者調整では、行政、事業者、住民などが目標を共有し、協力してハード対策（インフラ整備）やシステム対策（規制や料金施策）を実現することが求められる。

MMは「状況診断」と「取り組み」を繰り返すマネジメント・サイクルを通じて、持続的に展開される。一人ひとりの小さな行動変容の積み重ねが、渋滞緩和や公共交通活性化といった大きな成果につながる。

Wardropの第1原則（等時間原則）

Wardropの第2原則（総時間最小化原則）
システム最適配分の基本的概念：交通ネットワーク上の総費用（総所要時間）が最小となる
システム最適配分：社会全体にとって望ましい状態を示すための手法→規範モデル
利用者均衡配分は実際に生じる可能性のある交通状態を表現するための記述モデルである一方、システム最適配分は理想的な規範モデル

Frank-Wolfe法
1. リンク交通量を0として、各ODペアの最短経路上に、各ODペアの分布交通量のすべてを流して、初期リンク交通量xa^(0)を得る
2. xa^(n-1)に対して、リンク所要時間を更新する（ta^(n) = ta(xa^(n-1)を計算する）
3. 各ODペアの最短経路を求めて、最短経路上に、各ODペアの分布交通量のすべてを流して、リンク交通量yaを得る。
4. xa^(n) = αya + (1-α)xa^(n-1)として、αの一次元探索によって、目的関数Z(xa^(n))を最小にするようなαとxa^(n)を求める
5. 条件を満たせば計算を終了する。そうでなければn = n+1として2.に戻る

手段別交通量
交通手段別に分布交通量を求めること

重力モデル
qij = k*Xi^α*Yj^β/tij^γ
tij：ゾーンij間の交通抵抗（通常は所要時間や距離など）

分布交通量の推定手法
1. 現在のOD表からゾーンij間のOD交通量Tij、ゾーンiの発生交通量Xi、ゾーンjの集中交通量Yjを求める
2. 将来のゾーンiの発生交通量Ui、ゾーンjの集中交通量Vjを用いて、各ゾーンの発生・集中交通量の成長率(Fi, Gj)を求める。Fi = Ui / Xi, Gj = Vj / Yj
3. 分布交通量の第一次推定値を求める。qij = Tij * f(Fi, Gj)
4. 分布交通量の第1次推定値から、発生・集中交通量の第1次推定値Xi^(1), Yj^(1)を求める。Xi^(1) = Σj qij^(1), Yj^(1) = Σi qij^(1)
5. Xi^(1), Yj^(1)が、Ui, Vjに一致しているか確かめる。Fi^(1) = Ui / Xi^(1), Gj^(1) = Vj / Yj^(1)
6. 一致していれば終了する。そうでなければ3.に戻る

フレータ法
f = Fi^(k) * Gj^(k) * (Li^(k) + Mj^(k)) / 2
Li^(k) = Xi^(k) / (Σj qij^(k) * Gj^(k))
Mj^(k) = Yj^(k) / (Σi qij^(k) * Fi^(k))

デトロイト法
f = Fi^(k) * Gj^(k) / (Σj Vj / Σj Yj^(k))
